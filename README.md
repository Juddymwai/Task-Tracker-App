 ## Description
 <p>The work of the app is to help a user be  more organized. The app enables a user to create a schedule of the tasks or projects at hand. After completing a task, the user can mark off the task and move on to another. The app is secure and personal in that , no other person has access to another personâ€™s profile</p>
